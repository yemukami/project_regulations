# ベニバナ・サットプロジェクト ブランド・商標管理整備計画

## 現状分析

### 課題の特定
1. **「ベニバナ・サットプロジェクト」名称**: 商標未登録状態
2. **プロジェクトロゴ**: コアメンバー個人の著作権、講座への帰属未整理
3. **企業の独自使用**: 関連マーク・商品開発時の縛りなし
4. **規約の空白**: ブランド保護に関する規定なし

### リスク評価
- **高リスク**: 他社による商標先取り、ブランド価値の毀損
- **中リスク**: 企業間でのブランド使用競合、収益機会の逸失
- **低リスク**: 著作権侵害クレーム、混乱による教育効果減少

## 整備手順・実施計画

### **フェーズ1: 緊急対応（実施期間: 1-2ヶ月）**

#### **1-1. ロゴ著作権の整理**
**目的**: ロゴの権利関係明確化
**手順**:
1. **現状確認**（1週間）
   - ロゴ作成者の特定・連絡
   - 作成経緯・使用許諾状況の整理
   - 既存使用実績の調査

2. **権利移転手続き**（2週間）
   - 著作権譲渡契約書の作成
   - 作成者との協議・合意
   - 譲渡契約の締結

3. **権利登録**（1週間）
   - 著作権登録申請（任意）
   - 講座資産としての台帳整備

**担当**: 渉外部 + 法務相談
**成果物**: 著作権譲渡契約書、権利台帳

#### **1-2. 商標調査・出願準備**
**目的**: 商標権確保の基盤整備
**手順**:
1. **商標調査**（2週間）
   - 「ベニバナ・サット」「ベニバナ・サットプロジェクト」の類似商標検索
   - 関連分野（宇宙・衛星・教育）での登録状況確認
   - 弁理士への調査依頼

2. **出願戦略策定**（1週間）
   - 登録対象範囲の決定（名称・ロゴ・スローガン等）
   - 商標区分の選定（教育・研究・宇宙関連等）
   - 出願スケジュール・予算の確定

3. **出願準備**（1週間）
   - 商標出願書類の作成
   - ロゴデザインの確定・清書
   - 弁理士との最終確認

**担当**: 知財検討部会 + 外部弁理士
**成果物**: 商標調査報告書、出願戦略書、出願書類

### **フェーズ2: 制度整備（実施期間: 1-2ヶ月）**

#### **2-1. 規約へのブランド管理条項追加**
**目的**: 基本的なブランド保護ルールの明文化
**手順**:
1. **条項案作成**（1週間）
   ```markdown
   第○条（ブランド・商標管理）

   ○-1. プロジェクト名称・ロゴの管理
   1. 「ベニバナ・サットプロジェクト」名称の使用権はプログラム統括が管理
   2. プロジェクトロゴの著作権は講座に帰属
   3. 参画企業による使用は事前承認制とする

   ○-2. 関連ブランド開発の規制
   1. プロジェクト関連商品・サービスの商標登録は事前協議必須
   2. 独自関連マーク作成時はプロジェクトとの関係性を明示
   3. 商業利用時は講座への報告・協議義務を負う

   ○-3. 使用許可基準
   1. 教育目的・広報目的：原則許可
   2. 商業目的：個別審査・条件付き許可
   3. プロジェクト理念に反する使用：禁止

   ○-4. 違反時の措置
   1. 無許可使用発見時の是正要求権
   2. 重大な違反時の参画資格停止
   3. 損害発生時の賠償請求権
   ```

2. **コアチーム承認**（1週間）
   - 条項案の内容検討・修正
   - プログラム統括による最終承認
   - 既存規約への統合作業

3. **参画企業への説明**（2週間）
   - 規約改訂の事前通知
   - 改訂内容・影響の説明会開催
   - 企業からの意見聴取・調整

**担当**: PMO + 法務相談
**成果物**: 規約改訂版、説明資料

#### **2-2. Tier別合意文書への条項追加**
**目的**: 各Tier文書でのブランド使用ルール明確化
**手順**:
1. **Tier別条項案作成**（1週間）

   **Tier1用**:
   ```markdown
   第○条（ブランド使用権）
   1. プロジェクト名称・ロゴの共同使用権を付与
   2. 関連商品開発時の優先協議権を保有
   3. 商業利用収益の一部還元義務
   ```

   **Tier2用**:
   ```markdown
   第○条（ブランド使用）
   1. 支援企業としての表示におけるロゴ使用許可
   2. 独自商品開発時の事前協議義務
   ```

   **Tier3-4用**:
   ```markdown
   第○条（ブランド使用制限）
   1. 協力関係表示に限定したロゴ使用許可
   2. 商業利用は別途個別協議
   ```

2. **既存文書の更新**（1週間）
   - 各Tier合意文書テンプレートへの条項追加
   - 整合性チェック・文言調整

**担当**: 渉外部
**成果物**: 更新版Tier別合意文書テンプレート
