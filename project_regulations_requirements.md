# 山形大学超小型衛星開発プロジェクト規約策定 - 要件文書

## プロジェクト概要
**目的**: 財団寄付による山形大学講座内で実施される超小型衛星開発プロジェクトの運営規約策定

**背景**:
- 外部企業が手弁当で参加する大学内プロジェクト
- 教育目的と商業目的のバランス調整が必要
- 知財・権限・統制に関する明確なルール策定が急務

## 主要課題
1. **参加企業選定基準の不明確性** - 現在進行中のBBMフェーズで企業選定が発生中
2. **手弁当→外注移行ルールの未整備** - FMフェーズ（1年後）までに明確化必要
3. **知財・情報管理の複雑性** - オープンイノベーションと企業機密の両立
4. **教育vs商業目的の線引き** - 大学コンプライアンス対応

## プロジェクト基本情報
- **期間**: 2027年度末まで（打ち上げ目標）
- **予算**: 年5千万円（寄付講座独自予算）
- **組織規模**: コア8名、手弁当企業5社10名（BBM/EM）→7社20名（FM）
- **成果物**: 論文10本、特許取得、衛星製造・運用

## 規約構成
1. 基本方針（目的・理念・組織構成）
2. 参加企業管理（選定基準・移行ルール）
3. 知財・成果物管理（権利帰属・情報管理）
4. 学生関与ルール
5. 運営・責任分担

## 成功基準
- 大学コンプライアンス適合
- 企業参加意欲の維持
- 教育目的の担保
- オープンイノベーション実現
