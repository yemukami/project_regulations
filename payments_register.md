# 有償案件台帳（Payments Register）

調達の透明性確保のため、有償案件の要点を台帳として管理する。

| No | 件名 | ベンダー | 金額（税抜） | 方式（比較/単一） | 理由サマリ | 期間 | 承認者 | 契約日 | 紐付ログ |
|---:|---|---|---:|---|---|---|---|---|---|
| 1 |  |  |  |  |  |  |  |  | issue-# |

運用メモ：
- 「方式」は比較見積／ミニRFP／単一選定の別を記載。
- 「理由サマリ」は比較結果の要点又は単一選定理由の要点を記載。
- 「紐付ログ」はissues番号や選定記録ファイルへのリンクを記載。
